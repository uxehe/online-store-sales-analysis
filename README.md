# Анализ продаж интернет-магазина

## Описание проекта

Проект посвящён исследовательскому анализу данных о продажах интернет-магазина.  
Датасет содержит информацию о заказах, клиентах, товарах, категориях, регионах, способах доставки и сумме продаж.

## Цель проекта

Цель проекта — изучить структуру продаж, определить наиболее прибыльные категории товаров, проанализировать динамику продаж и выявить ключевые закономерности в данных.

## Используемые инструменты

- Python
- Pandas
- Matplotlib
- SQLite
- SQL
- Jupyter Notebook

## Данные

В проекте используется датасет `train.csv`, содержащий следующие признаки:

- `Order ID` — идентификатор заказа
- `Order Date` — дата заказа
- `Ship Date` — дата отправки
- `Ship Mode` — способ доставки
- `Customer ID` — идентификатор клиента
- `Segment` — сегмент клиента
- `City`, `State`, `Region` — география заказа
- `Category`, `Sub-Category` — категория и подкатегория товара
- `Product Name` — название товара
- `Sales` — сумма продаж

## Этапы проекта

1. Загрузка данных
2. Первичный обзор данных
3. Проверка пропусков и дубликатов
4. Приведение типов данных
5. Создание новых признаков
6. Расчёт ключевых метрик
7. Анализ продаж по времени
8. Анализ продаж по категориям и подкатегориям
9. Анализ продаж по регионам
10. Анализ клиентских сегментов
11. Анализ способов доставки
12. SQL-анализ
13. Формулирование итоговых выводов

## Основные выводы

- Общая сумма продаж составила 2 261 536.78.
- В данных 4 922 уникальных заказа, 793 клиента и 1 861 товар.
- Максимальные продажи наблюдаются в 2018 году — 722 052.02.
- Самая прибыльная категория — Technology, сумма продаж 827 455.87.
- Самый сильный регион по продажам — West, сумма продаж 710 219.68.
- Средний чек составил 459.48.
